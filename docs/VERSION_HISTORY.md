# Project Aether Version History

This document records major checkpoints in the Project Aether concept so earlier work remains visible as the project evolves.

## V2 — Interactive Thermodynamic Presentation

**Status:** Archived previous version  
**Location:** `versions/v2/index.html`  
**Purpose:** Preserve the first repo-ready standalone interactive HTML presentation.

### What V2 includes

- Standalone HTML, CSS, and JavaScript.
- Interactive thermodynamic engine.
- Cooling-mode presets: legacy air, hybrid, and immersion.
- Sliders for compute load, ambient temperature, passive facade engagement, waste-heat routing, and sunlight.
- Live outputs for PUE, grid draw, self-generation, water usage, homes heated, annual energy savings, annual cost savings, avoided CO2, and water savings.
- Animated SVG closed-loop heat-flow schematic.
- Design explanations for:
  - Passive double-skin facade.
  - Thermal chimney and wind-catcher concept.
  - Aether-Rack micro-climate cell.
  - Immersion cooling and waste-heat recycling loops.
  - Micro-grid, BIPV, thermal battery, and AI control system.
  - Individual action roadmap for advocacy, consumer pressure, and public education.

### Why V2 matters

V2 marks the transition from a written concept into an interactive presentation tool. It is not the final project, but it captures the point where Project Aether became demonstrable as a live model rather than just a proposal.
